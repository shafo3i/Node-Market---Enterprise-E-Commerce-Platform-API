Get CSRF Token

GET /api/csrf-token

Generates and returns a CSRF token for secure mutations.

Response:

{
  "csrfToken": "abc123def456..."
}
‚ö†Ô∏è Important: Include this token in the x-csrf-token header for all POST/PUT/PATCH/DELETE requests.

Usage Example:

// Fetch CSRF token
const response = await fetch('http://localhost:3003/api/csrf-token', {
  credentials: 'include'
});
const { csrfToken } = await response.json();

// Use in mutation request
await fetch('http://localhost:3003/api/cart/items', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-csrf-token': csrfToken
  },
  credentials: 'include',
  body: JSON.stringify({ productId: 'xxx', quantity: 1 })
});

üîí Security Features

CSRF Protection

‚úì Enabled

All mutation endpoints (POST/PUT/PATCH/DELETE) require CSRF token validation.

Exempt Paths:

  • /api/webhooks/* - Verified via signature
  • /api/auth/* - Handled by Better Auth
  • /api/csrf-token - Token generation endpoint
  • /api/track - Mobile tracking endpoint

Rate Limiting

‚úì Active
  • Window: 15 minutes
  • Max Requests: 500 per IP
  • Applies To: All endpoints

Security Headers (Helmet.js)

‚úì Configured
  • Content Security Policy (CSP): Strict resource loading
  • HSTS: 1-year max age with preload
  • X-Content-Type-Options: nosniff
  • X-XSS-Protection: enabled
  • Referrer Policy: same-origin

Session Management

  • Duration: 24 hours
  • httpOnly: true
  • secure: true (production)
  • sameSite: strict (production) / lax (development)

HTTPS Enforcement

‚úì Production

Automatic redirect from HTTP to HTTPS in production environment.

Using cURL

# Get welcome message

# Get CSRF token
curl -c cookies.txt http://localhost:3003/api/csrf-token

# Get current session
curl -b cookies.txt http://localhost:3003/api/me

Using JavaScript/TypeScript

// Fetch helper with CSRF
async function fetchWithCsrf(url, options = {}) {
  // Get CSRF token
  const tokenRes = await fetch('http://localhost:3003/api/csrf-token', {
    credentials: 'include'
  });
  const { csrfToken } = await tokenRes.json();
  
  // Make authenticated request
  return fetch(url, {
    ...options,
    headers: {
      ...options.headers,
      'x-csrf-token': csrfToken,
      'Content-Type': 'application/json'
    },
    credentials: 'include'
  });
}

// Usage
const response = await fetchWithCsrf('http://localhost:3003/api/cart/items', {
  method: 'POST',
  body: JSON.stringify({ productId: 'xxx', quantity: 1 })
});
